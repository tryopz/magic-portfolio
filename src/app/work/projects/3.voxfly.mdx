## Overview

Voxfly is an original flight simulator that combines realistic physics with procedurally generated worlds. Each playthrough offers a new challenge, as players must navigate evolving environments, avoid the deadly sun, and adapt to dynamic weather conditions. The project showcases advanced Unity development, procedural content generation, and engaging gameplay design.

## Key Features

- **Realistic Flight Physics**: Responsive controls and authentic flight dynamics.
- **Procedural Terrain**: Unique islands, caves, and plains generated for every session.
- **Dynamic Weather**: Turbulence and cloud cover evolve with player progress.
- **Environmental Hazards**: Sun exposure, water, and obstacles create strategic depth.
- **Skill-Based Scoring**: Points awarded for risky maneuvers and close calls.

## Technologies Used

- **Unity**: Game engine for physics, rendering, and procedural generation.
- **C#**: Gameplay logic, environment systems, and custom mechanics.
- **Shader Graph & VFX**: Advanced visual effects and atmospheric rendering.
- **Procedural Algorithms**: Terrain and weather generation.

## Project Highlights

- Designed for replayability and skill mastery, with each flight offering new challenges.
- Demonstrates expertise in Unity, C#, and real-time procedural content.
- Developed in a collaborative team setting, emphasizing both technical and creative skills.
